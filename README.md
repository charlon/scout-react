# scout-react
ReactJS rewrite of Scout.

### Prerequisites
* Node
* ReactJS
* Create-React-App (used to scaffold this application)

### Installation

Clone this repository:
```
  $ git clone https://github.com/charlon/scout-react.git
  $ cd scout-react
```

Install Node dependencies:
```
  $ npm install
```
Start the server (webpack):
```
  $ npm start
```
